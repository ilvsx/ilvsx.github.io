Magento

Notes | Jul 4, 2016

选择版本

在首页上能看到 Magento 分为三个版本：商业云版本，商业版，社区版本，前两个版本都需要和 Magento 联系才能获取到，这里我们选择第三个社区版本，也就是 Magento Community Edition 2.1

环境要求

  • Linux x86-64
  • Composer ：php 组件的依赖管理工具，并且可以生成符合 PSR-4 标准的自动加载器
  • Web Server
    • Apache 2.2 or 2.4
    • Nginx 1.8.x 或者最新稳定版本
  • PHP
    • php 版本：7.0 或者 .5.6 或者 5.5
    • 必要的扩展： bc-math (Enterprise Edition only), curl, gd, ImageMagick 6.3.7, intl, mbstring, mcrypt, mhash, openssl, PDO_MySQL, SimpleXML, soap, xml, xsl, zip
    • 非必要但是建议的扩展
      • opcache ：可以缓存编译每个 php 脚本得到的操作码，加速这个处理过程
      • php_xdebug ：php 分析器之一，能轻易找出瓶颈和性能问题，注意生产环境中不要启用，它对性能和资源的消耗影响很大，生产环境中请用 XHProf
    • 配置文件
      • safe_mode off ：关闭 safe_mode
      • memory_limit minimum 512 MB ：运行时的内存大小限制，最小 512 MB
  • DataBase
    • MySQL 5.6.x
  • SSL
  • Mail Server

下载

  • 打开下载页面
  • 可以看到有多个选择，这里我们选择第二个 Full Release with Sample Data ，它包含了 Magento 的最新版本和数据样例
  • 下载完之后解压
