为什么是 Linux 而不是 OSX 以及 Windows?

  • Windows
    • 优点：娱乐性特别强，如果你必须要使用 QQ，或者需要高质量视频播放(Lav + ffdshow + SVP即时帧插补)、海量的游戏、还有丰富的软件，那就只有 Windows 可选了，特别是有的软件只有 Windows 版本，以及某些软件在 Windows 下表现特别好
    • 缺点：毫无技术学习上的长期投资性可言，Xp 时代折腾注册表、GHOST，解决过各种蓝屏系统慢图标消失重启后没声音等以及各种奇葩问题，现在看来这些折腾过的东西毫无价值，都不值得一提，还不如直接重装系统来的省事直接
  • OSX
    • 优点：开箱即用，颜值高，软件质量普遍较高，和 Linux 相比至少能有好用的 QQ
    • 缺点：缺少娱乐性，这点其实也算是优点，因为这样你可以全身心的投入到工作中
  • Linux
    • 优点：对于开发人员来说极度友好
    • 缺点：对应 Windows 的所有优点，如果不是开发人员就别考虑 Linux 了

为什么是 Ubuntu 而不是 Arch 等其他发行版本？

比如： 原因就一点开箱即用，少折腾，可以平滑地从 OSX 切换到 Linux

安装系统

这是最方便以及最省心地方法：

  • 在京东上购买16G USB3.0 的U盘 选择 16G 而不是更大容量的 U盘是因为没必要，U盘不会也不应该存放长期存储的文件，我们需要的是一个随时可以格式化的U盘。
  • 下载 Ubuntu 安装镜像 Download Ubuntu Desktop
  • 使用 Rufus 制作引导盘 How to create a bootable USB stick on Windows
  • 后续安装一路下一步即可
